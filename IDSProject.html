AI-Based Intrusion Detection System (IDS) Project

This page provides an overview of my AI-based Intrusion Detection System (IDS) project. The project leverages machine learning techniques to analyze network traffic and detect anomalous behavior that may indicate potential cyber threats.

Key Features

  • Built using Python and Scikit-learn
  • Uses the UNSW-NB15 dataset for training/testing
  • Random Forest + RandomizedSearchCV for anomaly detection
  • Planned integration with Suricata or Snort for real-time alerts
  • Deployed in a homelab environment via Ubuntu VM

Benefits

This project demonstrates practical cybersecurity and data science skills, including dataset preprocessing, model training, and deployment in a SOC-like environment.
