Valence College Fee Invoice (Java)

Overview

Coursework • OOP • CLI/Console

A Java application that generates fee invoices for different student types (PhD, MS, Undergraduate), using object‑oriented design with class hierarchies and polymorphism. Supports input validation, tuition calculation rules, and formatted invoice output.

Key Features

  • Class hierarchy: Student base class with PhDStudent, MsStudent, and UndergraduateStudent subclasses.
  • Encapsulated tuition/fee rules per student type with overridable methods.
  • Robust input parsing and formatted invoice summaries.

Skills Gained

Java OOP (Inheritance/Polymorphism) Data Validation Formatting & I/O Unit Testing Mindset

InventoryManager / NileDotCom (Java Swing)

Overview

Coursework • GUI • CSV

A desktop order‑entry and inventory tracking tool built with Java Swing. Reads products from a CSV file, lets users add items to a cart, calculates totals/taxes/discounts, and exports transactions for record‑keeping.

Key Features

  • CSV import/export for inventory and transactions (BufferedReader, FileWriter).
  • Form validation, clear/reset of fields, and running order totals.
  • Separation of concerns with Item, Order, and manager/controller classes.

Skills Gained

Java Swing File I/O (CSV) Event‑Driven Programming MVC Thinking Exception Handling
